# 社团管理系统简介 java1149

## 项目概述

社团管理系统是一款基于B/S架构的Web应用，采用MVC模式设计，实现前后端分离。系统致力于为各类社团提供一个便捷、高效的管理平台。

## 技术栈

- **开发语言**：Java
- **数据库**：Mysql
- **后端框架**：SpringBoot
- **持久层框架**：Mybatis
- **前端框架**：Vue
- **开发工具**：IDEA、JDK1.8、Maven

## 系统角色

- **用户**：普通登录用户，可进行基本的信息管理。
- **超级管理员**：拥有最高权限，可以管理所有社团。
- **社团管理员**：负责本社团的管理活动，包括信息管理、活动管理等。
- **学生用户**：参与社团活动，进行社团信息管理。

### 功能模块

- **用户模块**
  - 登录/注册
  - 用户信息管理

- **超级管理员**
  - 所有社团管理

- **社团管理员**
  - 社团信息管理
  - 社管活动管理
  - 入团申请处理
  - 费用管理
  - 社团活动通知发布

- **学生用户**
  - 社团信息查看
  - 社团活动参与
  - 入团申请
  - 费用缴纳
